# This is a fun way to declare your love to your Valentine

► You can see the live result @ https://yannyny.github.io/Valentine/

► You can customize it by modifying:
  - the h1 header and the buttons in [`index.html`](https://github.com/Yannyny/Valentine/blob/main/index.html)
  - the background emojis, the h3 header, the paragraphs and the gif link in [`yes.html`](https://github.com/Yannyny/Valentine/blob/main/yes.html)
  - the random movement of the 'No' button in [`script.js`](https://github.com/Yannyny/Valentine/blob/main/script.js)
  - the CSS styles in [`styles.css`](https://github.com/Yannyny/Valentine/blob/main/style.css)
